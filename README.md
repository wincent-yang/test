# test 项目

这是一个 Python 教学练习仓库，包含若干示例与习题文件，主要用于学习基础语法与面向对象编程。

目录结构
- DIY\: 各类示例与练习（例如 demo1.py、示例2-01 等）
- chap2\: 第二章示例与习题（面向对象、时钟示例等）

运行要求
- Python 3.x

运行示例
- 在仓库根目录执行：
  python DIY\demo1.py
  python chap2\1.py
  python DIY\示例2-01.py

说明
- 部分文件名含中文，运行时请使用完整路径或在支持的终端中复制粘贴文件名。
- 练习说明见 DIY\note.txt
